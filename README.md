# ROI Analysis Console Application

ROI Analysis Console Application is a Java project that collects equipment data, calculates return on investment (ROI), and ranks items from highest ROI to lowest ROI. The program uses a console-based interface and stores equipment records in a linked list for analysis and sorted output.

## Project Overview

This project was created as a final assignment for a Java course. Its purpose was to apply object-oriented programming and data structure concepts to a practical business problem by evaluating equipment investments based on ROI.

The application prompts the user to enter equipment details such as name, gain, and cost, then calculates ROI for each item and displays a ranked list based on performance.

## Tech Used

- Java

## Features

- Accepts user input for equipment name, gain, and cost
- Stores equipment records in a linked list
- Calculates ROI using a business formula
- Sorts equipment from highest ROI to lowest ROI
- Displays ranked results in the console

## What I Built

I built the program logic to store records in a data structure, calculate ROI values, sort the results, and display the ranked output. This project helped strengthen my understanding of working with data structures and translating business logic into software.

## File Structure

- `Main.java` — handles user input, ROI calculation, sorting, and output
- `Equipment.java` — defines the equipment object and related fields
